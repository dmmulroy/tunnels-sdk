Implement the `IngressManager` on the `Tunnel` object for managing ingress rules.

## Deliverables
- `src/managers/ingress.ts` — `IngressManager` class
- Methods: `list()`, `add(rule)`, `remove(hostname)`, `set(rules[])`
- Auto-appends catch-all if missing when setting rules
- Uses CF API: PUT `/accounts/{id}/cfd_tunnel/{id}/configurations`

## API
```ts
const rules = await tunnel.ingress.list()
await tunnel.ingress.add({ hostname: "new.example.com", service: "http://localhost:9090" })
await tunnel.ingress.remove("old.example.com")
await tunnel.ingress.set([...rules])
```
