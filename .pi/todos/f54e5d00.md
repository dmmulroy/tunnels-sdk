Implement the `cloudflared` binary management module. This is the foundation — every tunnel operation needs the binary.

## Deliverables
- `src/bin/cloudflared.ts` — main module
- Platform detection (darwin-amd64, darwin-arm64, linux-amd64, linux-arm64, windows-amd64)
- Download from GitHub releases (`cloudflare/cloudflared`)
- Cache in `node_modules/.cache/tunnel-sdk/bin/`
- Version-lock to a specific cloudflared release (pinned in code)
- `install()`, `update()`, `remove()`, `isInstalled()` methods
- `path` and `version` getters
- Checksum verification after download
- Tests with mocked HTTP (no real downloads in CI)

## Exported API
```ts
import { cloudflared } from "tunnel-sdk/bin"
cloudflared.path       // string
cloudflared.version    // string
await cloudflared.install()
await cloudflared.install({ version: "2025.2.0" })
await cloudflared.update()
await cloudflared.remove()
await cloudflared.isInstalled() // boolean
```
