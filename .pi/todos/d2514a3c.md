Implement `DnsManager` on the `Tunnel` object for managing DNS CNAME records.

## Deliverables
- `src/managers/dns.ts` — `DnsManager` class
- Methods: `ensure(hostname, opts?)`, `remove(hostname)`, `list()`
- Zone detection: extract zone from hostname, find zone ID via CF API
- Idempotent `ensure()` — creates if missing, no-ops if exists and correct
- Uses CF API:
  - GET `/zones?name={zone}` — find zone ID
  - GET `/zones/{id}/dns_records?type=CNAME&name={hostname}` — check existing
  - POST `/zones/{id}/dns_records` — create
  - DELETE `/zones/{id}/dns_records/{id}` — remove

## API
```ts
await tunnel.dns.ensure("app.example.com")
await tunnel.dns.ensure("app.example.com", { proxied: true })
await tunnel.dns.remove("old.example.com")
const records = await tunnel.dns.list()
```
