Implement the `expose(port)` function — the simplest path. Anonymous quick tunnel, no auth needed.

## Deliverables
- `src/expose.ts` — `expose(port, options?)` function
- Spawns `cloudflared tunnel --url localhost:{port}` as child process
- Parses stdout/stderr for the assigned `*.trycloudflare.com` URL
- Returns a `QuickTunnel` object with `.url`, `.close()`, `[Symbol.asyncDispose]()`
- Explicit Resource Management support (`await using tunnel = await expose(3000)`)
- Options: `protocol`, `hostname` (for authenticated tunnels — wired later)
- Process cleanup on `.close()`, SIGINT, SIGTERM, uncaught exception
- Integration test that actually exposes a port (skip in CI without binary)
- Unit tests for URL parsing, process lifecycle

## Exported API
```ts
import { expose } from "tunnel-sdk"

const tunnel = await expose(3000)
console.log(tunnel.url)
await tunnel.close()

// Or with `using`
await using tunnel = await expose(3000)
```
