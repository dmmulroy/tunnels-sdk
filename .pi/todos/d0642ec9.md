Implement `RouteManager` on `Tunnel` and `VNetManager` on `TunnelClient`.

## Deliverables
- `src/managers/routes.ts` — `RouteManager` class
  - `add(cidr, opts?)`, `remove(cidr)`, `list()`, `check(ip)`
  - CF API: `/accounts/{id}/teamnet/routes`
- `src/managers/vnets.ts` — `VNetManager` class
  - `create(name, opts?)`, `delete(name)`, `list()`
  - CF API: `/accounts/{id}/teamnet/virtual_networks`

## API
```ts
await tunnel.routes.add("172.16.0.0/16")
await tunnel.routes.add("10.0.0.0/8", { vnet: "production" })
await tunnel.routes.remove("172.16.0.0/16")
const routes = await tunnel.routes.list()
const check = await tunnel.routes.check("172.16.5.42")

await client.vnets.create("production")
await client.vnets.delete("staging")
const vnets = await client.vnets.list()
```
