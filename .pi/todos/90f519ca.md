## Goal

Rewrite `packages/tunnels/src/index.ts` as a thin async/await wrapper around the Effect SDK. Consumers who `import { TunnelClient } from "tunnels"` get the same familiar API — no Effect knowledge required. Internally it uses `ManagedRuntime` to bridge Effect programs to Promises.

## Context

The current `src/index.ts` re-exports classes and functions. The new version creates a `ManagedRuntime` from the `LiveLayer` and exposes async/await methods that delegate to the Effect services.

**Key design change**: The old `Tunnel` class was a mutable object with nested managers (`tunnel.ingress.add(...)`, `tunnel.dns.ensure(...)`). The new `TunnelInfo` is an immutable data object. Sub-resource operations move to the client level: `client.ingress.add(tunnelId, ...)`. This is a breaking change in the wrapper API.

**`TunnelClient` class**:
```ts
export interface TunnelClientOptions {
  accountId: string
  apiToken: string
  baseUrl?: string
}

export class TunnelClient {
  private readonly runtime: ManagedRuntime.ManagedRuntime<...>

  readonly tunnels: TunnelClientTunnels
  readonly ingress: TunnelClientIngress
  readonly dns: TunnelClientDns
  readonly routes: TunnelClientRoutes
  readonly vnets: TunnelClientVNets

  constructor(options: TunnelClientOptions) {
    const config = new CloudflareApiConfig({
      accountId: options.accountId,
      apiToken: Redacted.make(options.apiToken),
      baseUrl: options.baseUrl,
    })
    this.runtime = ManagedRuntime.make(LiveLayer(config))
    this.tunnels = new TunnelClientTunnels(this.runtime)
    this.ingress = new TunnelClientIngress(this.runtime)
    this.dns = new TunnelClientDns(this.runtime)
    this.routes = new TunnelClientRoutes(this.runtime)
    this.vnets = new TunnelClientVNets(this.runtime)
  }

  async dispose(): Promise<void> {
    await this.runtime.dispose()
  }
}
```

**`expose()` wrapper**:
```ts
export async function expose(port: number): Promise<{
  readonly url: string
  close(): Promise<void>
  [Symbol.asyncDispose](): Promise<void>
}> {
  // Create a dedicated runtime with just CloudflaredBinary + NodeServices
  const runtime = ManagedRuntime.make(
    Layer.mergeAll(CloudflaredBinary.layer, NodeServices.layer)
  )

  // Run expose in a scope, keep the scope open
  // When close() is called, dispose the runtime (which closes the scope, killing the process)
  const result = await runtime.runPromise(
    exposeEffect(port).pipe(Effect.scoped)
  )

  return {
    url: result.url,
    close: () => runtime.dispose(),
    [Symbol.asyncDispose]: () => runtime.dispose(),
  }
}
```

## Effect patterns reference

```ts
// ManagedRuntime bridges Effect → Promise
const runtime = ManagedRuntime.make(myLayer)
const result = await runtime.runPromise(myEffect)
await runtime.dispose() // cleanup

// Service accessor pattern
```
